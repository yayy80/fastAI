# Usage:
1. Clone this repository
2. Run "cd chatbot"
3. Run "python chat.py"
4. Wait, then type something
5. Wait for it to finish
6. You will see "textengine.py" in action
7. Wait for it to finish, and see it appear in "saved_responses.txt"
